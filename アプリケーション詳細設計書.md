#### 処理15：予定在庫取得

- **クラスID**: `StockPlanNumSelectLogic`
- **クラス名**: レンタルオーダー入力（入荷予定入力）検索ロジック
- **メソッドID**: `getPlanStockQty`
- **メソッド名**: 予定在庫取得
- **検索方法**: 外出しSQL

##### パラメータ

| No | パラメータ | 設定値 |
|----|-----------|--------|
| 1 | 拠点CD | パラメータ.拠点CD |
| 2 | 品目CD | パラメータ.品目CD |
| 3 | 希望日 | パラメータ.レンタル希望日 |

##### 予定在庫の算出

＜予定在庫＞は以下の5つのサブクエリで算出します。

```
現在庫数① + 予定レンタル数② + 予定入庫数③ - 予定返却数④ - 予定出庫数⑤
```

| サブクエリ | 対象テーブル | SUM対象 |
|-----------|------------|--------|
| ①現在庫数 | 在庫 | SUM（在庫.引当可能数） |
| ②予定レンタル数 | 入荷予定ボディ | SUM（入荷予定ボディ.実績数）、入荷ステータス=05:実績待ち |
| ③予定返却数 | 出荷指示ボディ | SUM（出荷指示ボディ.指示数）、出荷ステータス=05:実績待ち |
| ④予定入庫数 | 在庫移動指示ボディ | SUM（在庫移動指示ボディ.指示数）、入出庫区分=0:入庫 |
| ⑤予定出庫数 | 在庫移動指示ボディ | SUM（在庫移動指示ボディ.指示数）、入出庫区分=1:出庫 |

##### サブクエリ詳細

**サブクエリ①：現在庫数**

- メインテーブル：在庫
- 取得項目：在庫.ロケーションID、在庫.商品ID、SUM（在庫.引当可能数）
- 結合テーブル：商品マスタ（商品ID）、拠点マスタ（ロケーションID）
- 検索条件：在庫.削除フラグ='0'、商品マスタ.商品CD=パラメータ.拠点CD、商品マスタ.削除フラグ='0'、拠点マスタ.拠点CD=パラメータ.商品CD、拠点マスタ.削除フラグ='0'
- グルーピング：在庫.ロケーションID、在庫.商品ID

**サブクエリ②：予定レンタル数**

- メインテーブル：入荷予定ボディ
- 取得項目：入荷予定ボディ.予定ロケーションID、入荷予定ボディ.予定商品ID、SUM（入荷予定ボディ.実績数）
- 結合テーブル：入荷予定ヘッダ（入荷予定ヘッダID）
- 検索条件：入荷予定ボディ.入荷ステータス = '05:実績待ち'（2023/1/9 HDIS修正）、入荷予定ボディ.入出庫日 <= パラメータ.希望日、入荷予定ボディ.削除フラグ='0'、入荷予定ヘッダ.削除フラグ='0'
- グルーピング：入荷予定ボディ.予定ロケーションID、入荷予定ボディ.予定商品ID

**サブクエリ③：予定返却数**

- メインテーブル：出荷指示ボディ
- 取得項目：出荷指示ボディ.指定ロケーションID、出荷指示ボディ.予定商品ID、SUM（出荷指示ボディ.指示数）
- 結合テーブル：出荷指示ヘッダ（出荷指示ヘッダID）
- 検索条件：出荷指示ボディ.出荷ステータス = '05:実績待ち'（2023/1/9 HDIS修正）、出荷指示ボディ.入出庫日 <= パラメータ.希望日、出荷指示ボディ.削除フラグ='0'、出荷指示ヘッダ.削除フラグ='0'
- グルーピング：出荷指示ボディ.指定ロケーションID、出荷指示ボディ.予定商品ID

**サブクエリ④：予定入庫数**

- メインテーブル：在庫移動指示ボディ
- 取得項目：在庫移動指示ボディ.ロケーションID、在庫移動指示ボディ.在庫商品ID、SUM（在庫移動指示ボディ.指示数）
- 結合テーブル：在庫移動指示ヘッダ（在庫移動指示ヘッダID）
- 検索条件：在庫移動指示ステータス='00:未作業'、入出庫区分='0:入庫'、入庫日 <= パラメータ.希望日、削除フラグ='0'（ボディ・ヘッダ両方）
- グルーピング：在庫移動指示ボディ.ロケーションID、在庫移動指示ボディ.在庫商品ID

- メインテーブル：在庫移動指示ボディ
- 取得項目：在庫移動指示ボディ.ロケーションID、在庫移動指示ボディ.在庫商品ID、SUM（在庫移動指示ボディ.指示数）

## 3. 参考：予定在庫取得のSQLイメージ

```sql
select IFNULL(main.Qty + sub1.Qty + sub2.Qty + sub3.Qty + sub4.Qty,0) AS planStockQty -- 予定在庫数
from 
-- ①現在庫数を取得するメインクエリ
(
  select 
    ts.LOCATION_ID as LOCATION_ID,
    ts.PRODUCT_ID as PRODUCT_ID,
    SUM(IFNULL(ts.CHARGE_NUM,0)) AS Qty
  from t_stock ts
  left join m_product mp
    on mp.PRODUCT_ID = ts.PRODUCT_ID
  left join m_location ml
    on ml.LOCATION_ID = ts.LOCATION_ID
  where ts.DEL_FLG = 0
    and mp.PRODUCT_CD = 'PD01'
    and mp.DEL_FLG = 0
    and ml.LOCATION_CD = 'A0101'
    and ml.DEL_FLG = 0
  group by 
    ts.LOCATION_ID,
    ts.PRODUCT_ID
) main 
-- ②予定レンタル数を取得するサブクエリ
left join
(
  select 
    trpb.PLAN_LOCATION_ID as LOCATION_ID,
    trpb.PLAN_PRODUCT_ID as PRODUCT_ID,
    SUM(IFNULL(trpb.RESULTS_NUM,0)) AS Qty
  from t_receive_plan_b trpb
  inner join t_receive_plan_h trph
    on trpb.RECEIVE_PLAN_H_ID = trph.RECEIVE_PLAN_H_ID
  where (
    trpb.RECEIVE_STATUS in ('04','05')
    and trpb.IN_OUT_DT <= '20220906'
    and trpb.DEL_FLG = '0'
    and trph.DEL_FLG = '0'
  )
  group by 
    trpb.PLAN_LOCATION_ID,
    trpb.PLAN_PRODUCT_ID
) sub1
on sub1.LOCATION_ID = main.LOCATION_ID
and sub1.PRODUCT_ID = main.PRODUCT_ID
-- ③予定返却数を取得するサブクエリ
left join
(
  select 
    tsib.LOCATION_ID as LOCATION_ID,
    tsib.PLAN_PRODUCT_ID as PRODUCT_ID,
    SUM(IFNULL(tsib.INST_NUM,0)) AS Qty
  from t_shipping_inst_b tsib
  inner join t_shipping_inst_h tsih
    on tsib.SHIPPING_INST_H_ID = tsih.SHIPPING_INST_H_ID
  where (
    tsib.SHIP_STATUS in ('04','05')
    and tsib.IN_OUT_DT <= '20220906'
    and tsib.DEL_FLG = '0'
    and tsih.DEL_FLG = '0'
  )
  group by 
    tsib.LOCATION_ID,
    tsib.PLAN_PRODUCT_ID
) sub2
on sub2.LOCATION_ID = main.LOCATION_ID
and sub2.PRODUCT_ID = main.PRODUCT_ID
-- ④予定入庫数を取得するサブクエリ
left join
(
  select 
    tmib.LOCATION_ID as LOCATION_ID,
    tmib.PRODUCT_ID as PRODUCT_ID,
    SUM(IFNULL(tmib.INST_NUM,0)) AS Qty
  from t_move_inst_b tmib
  inner join t_move_inst_h tmih
    on tmib.MOVE_INST_H_ID = tmih.MOVE_INST_H_ID
  where 
    tmib.INOUT_TYPE in ('0')
    and tmib.MOVE_INST_STATUS = '00'
    and tmib.STORE_DT <= '20220906'
    and tmib.DEL_FLG = '0'
    and tmih.DEL_FLG = '0'
  group by 
    tmib.LOCATION_ID,
    tmib.PRODUCT_ID
) sub3
on sub3.LOCATION_ID = main.LOCATION_ID
and sub3.PRODUCT_ID = main.PRODUCT_ID
-- ⑤予定出庫数を取得するサブクエリ
left join
(
  select 
    tmib.LOCATION_ID as LOCATION_ID,
    tmib.PRODUCT_ID as PRODUCT_ID,
    SUM(IFNULL(tmib.INST_NUM,0)) AS Qty
  from t_move_inst_b tmib
  inner join t_move_inst_h tmih
    on tmib.MOVE_INST_H_ID = tmih.MOVE_INST_H_ID
  where (
    tmib.INOUT_TYPE in ('1')
    and tmib.MOVE_INST_STATUS = '00'
    and tmib.STORE_DT <= '20220906'
    and tmib.DEL_FLG = '0'
    and tmih.DEL_FLG = '0'
  )
  group by 
    tmib.LOCATION_ID,
    tmib.PRODUCT_ID
) sub4
on sub4.LOCATION_ID = main.LOCATION_ID
and sub4.PRODUCT_ID = main.PRODUCT_ID
;
```
